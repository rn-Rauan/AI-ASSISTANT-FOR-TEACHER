# AI-ASSISTANT-FOR-TEACHER
API e Frontend para um sistema inteligente que gera materiais didáticos (planos de aula, atividades avaliativas) de Cultura Digital. Utiliza IA generativa e dados estáticos da BNCC e Diretrizes do MEC para garantir alinhamento pedagógico. Projeto do Hackathon TADS IFPI 2026.
